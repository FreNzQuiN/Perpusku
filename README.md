# Perpusku

**Sistem Manajemen Perpustakaan** — Library management system built with Laravel 12.

## Quick Start

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm install && npm run build
php artisan serve
```

## Tests

```bash
php artisan test          # 119 tests, all pass
```
